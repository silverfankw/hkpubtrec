# A convenient tool to record public record journey in Hong Kong

![Static Badge](https://img.shields.io/badge/Click%20for-Project%20Demo-blue?link=https%3A%2F%2Fsilverf.dev%2Fpubtrec)

Using this tool, You can 
1. Mark your journey starting point and destination according to your selected public transport type and route.
   <img width="537" height="452" alt="msedge_ktw1thB067"  src="https://github.com/user-attachments/assets/24605649-214d-47e9-8edd-9547ac38da96" />

2. Record passenger numbers, note in predefined format (HHMM+x-y/z) for each stop during the journey.
	* **HHMM**: 24hours format for time representation
	* **x**: number of passengers getting on at that stop
	* **y**: number of passengers alighting at that stop
	* **z**: remark for that stop (optional)
  
   <img width="596" height="278" alt="msedge_eqO5rGTwUx" src="https://github.com/user-attachments/assets/79061b17-db8c-49ee-a901-677e38b9a9b3" />


3. List view to browse your past journey history, and display detail after selection.
	<img width="705" height="133" alt="msedge_ymVfg4xPsU" src="https://github.com/user-attachments/assets/18b88779-2e76-46f8-8656-13b6649e47f5" />

4. Export the journey info into image for sharing / JSON format for re-import.
	<img width="533" height="428" alt="journey-20260401-971" src="https://github.com/user-attachments/assets/8e43daf3-4b2f-4a59-a2cc-40108d9f26df" />

5. Responsive layout is available for easier data input in mobile devices.
